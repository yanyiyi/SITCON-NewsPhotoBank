# Hour of Code - Scratch 秀自己 由SITCON與政治大學合辦的「Hour of Code - Anycode can code」系列，第三場「Scratch，秀自己」在今天 (2014.12.13) 圓滿落幕。這次活動是由 code.org 在美國發起，鼓勵全球師生共同響應，使程式設計跳脫資訊相關背景，成為跨越男女老少的一項全民運動。 我們從報章雜誌中可以發現，隨著資訊時代到來，人們對於未來的想像，往往挾著資訊的創新與突破，造就時代一次又一次的跳躍。在這個時代，學習資訊越來越重要，即使這不代表每個人都必須成為工程師，但「資訊思維」確實挑戰著傳統分工與運作模式，一陣暗潮正一點一滴改造社會。藉由響應 Hour of Code ，SITCON期望能落實資訊教育向下扎根，並拓廣結合不同領域，激盪無限的想像。 這次的活動參與者年齡層分布很廣，從最小的3歲半到大學畢業，不分男女老少中青共同創作人生第一個程式。上半場，藉由 code.org 所提供的 Frozen 教材，以往初學者學習時，時常困惑的 if, for, function 等等，透過 Scratch 直覺式快速學習程式邏輯，使這些概念不再那麼生澀難懂。再結合流行動畫電影Frozen，生動有趣地將程式進行視覺化，參與者能迅速了解程式運行，並進行除錯。同時，在最後一關的自由創作，讓參與者就自己的創意，繪製一幅專屬於自己的畫。 而下半場請來了 Open StreetMap (OSM) 社群的詹為翔為我們分享。原本就攻讀公共衛生的他，畢業後服外交役，到了落後國家服務。為了追蹤病人後續的病情，他們時常必須穿越叢林，往來部落之間，交通頓時成為了一大麻煩。當地人認路往往憑著幾個簡單的路標，甚至是向某棵大榕樹下一個老人問路，有時這些令我們感到不可思議的方法真能奏效，卻有時候連當地人都難免會迷失在叢林之中，他們曾經嘗試過 Google Map ，但是卻發現離開都市地區後，阡陌交通不如都市般鉅細靡遺。最後決定引進「Open StreetMap」，透過眾人協力，並教導黑人使用這套工具，透過GPS定位，眾人協力合作為當地建立一套交通資訊系統。
