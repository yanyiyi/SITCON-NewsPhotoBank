「HackGen駭客世代」號召青年學子，透過對生活的觀察，將想法轉化為程式碼，進而實現的程式馬拉松（黑客松 Hackathon）。延續著 SITCON 理念，鼓勵學生利用程式改造校園，並且希望能夠在學生族群推廣，幫助更多人踏入開發的領域。在 HackGen 活動，活動過程密集集中於兩天之內，考驗著團隊內的討論以及分工，例如：確定產品定位、多人開發等等，對於沒有碰觸過的新鮮人而言，是一個門檻。因此在會前一個月，決定開一個工作坊進行教學。 這次由 Denny (黃一晉) 帶來的git。在HackGen過程中，團隊勢必會有一連串的分工合作，而談到多人開發，就不得不談一談 git 分散式版本控制系統。從 git 的基本檔案管理到最引以為傲的 branch ，透過操作以及實際開發專案經驗分享，使在場的學員能夠深入淺出，並且了解到實際開發可能碰到的問題。 晚上緊接著是會前。為了打破校園內外資訊的藩籬，鼓勵更多人投入專案開發，會前交流提供各方人馬彼此結束的機會，除了彼此專案構思討論之外，亦可以讓生手與一些經驗老到的開發者聊天，不論是校園內的學科生活、未來出路、第一次開發的經驗分享等等，促進參與者的交流。
